How to write content

For panel/plain modes, you can use:

\paragraph{Teach the pattern before teaching every symbol.}

\begin{itemize}
\item First key point
\item Second key point
\end{itemize}

\begin{enumerate}
\item A
\item B
\end{enumerate}

\begin{card}{Key idea}
Each neuron turns an incoming sensitivity into an outgoing sensitivity.
\end{card}

For equations, use:

\begin{equation}
\nabla_w L &= \frac{\partial L}{\partial z}\frac{\partial z}{\partial w}\\
&= \frac{\partial L}{\partial z}\frac{\partial z}{\partial w}
\end{equation}

For pseudocode, paste plain indented pseudocode.

For pseudocode-latex, paste indented pseudocode and write math as \( ... \) or \[ ... \].

Use Add figure in the block editor to insert images or figure-maker diagrams at the cursor position.
